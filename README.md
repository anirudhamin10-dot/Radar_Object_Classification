# Pattern Recognition Project: Radar Classification

This project classifies radar objects using SVM and Random Forest.

## How to Run

1.  **Install Libraries:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Get Data:**
    * Download the dataset from `https://zenodo.org/records/4559821`.
    * Create a folder named `data` in this directory.
    * Place the dataset sequence folders (containing `radar_data.h5` files) inside the `data` folder.

3.  **Run Code:**
    ```bash
    python Main.py
    ```
